# Автоматизация Excel — объединение отчётов с аналитикой

**Что делает:** Объединяет несколько Excel-файлов, создаёт сводные таблицы и аналитику.

## 🎯 Для чего нужно бизнесу

| Задача | Решение |
|--------|---------|
| Объединить ежемесячные отчёты продаж | Автоматически собирает все файлы из папки |
| Посчитать выручку по месяцам | Создаёт сводную таблицу |
| Узнать, сколько строк из каждого файла | Добавляет колонку с источником |
| Получить отчёт об обработке | Создаёт отдельный лист с метаданными |

## 🚀 Быстрый старт

```bash
pip install pandas openpyxl
python excel_analyzer.py
```
## 📊 Результат
**После запуска создаётся файл итоговая_аналитика.xlsx с 4 листами:**

- **Все данные** — объединённая таблица
- **Сводка по месяцам** — аналитика продаж
- **Статистика по файлам** — сколько строк из каждого источника
- **Отчёт** — дата обработки и параметры

## 🔧 Настройка
```bash
# Изменить папку с файлами
merge_and_analyze_excel("путь/к/папке")

# Изменить имя выходного файла
merge_and_analyze_excel("папка", "мой_отчёт.xlsx")
```
